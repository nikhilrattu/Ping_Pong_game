A classic Pong arcade game recreated using Python Turtle Graphics and Object-Oriented Programming.  
Two players control paddles to bounce the ball and score points against each other.

---

## Features

- Two-player gameplay
- Real-time paddle controls
- Ball movement and collision physics
- Dynamic ball speed
- Score tracking system
- Paddle and wall collision detection
- Modular OOP-based architecture

---

## Concepts Used

- Object-Oriented Programming (OOP)
- Python Turtle Graphics
- Collision Detection
- Keyboard Event Handling
- Game Loop Management
- Modular Programming Structure
- Score Tracking System

---

## Project Structure

| File | Purpose |
|------|----------|
| `main.py` | Controls the game loop and execution |
| `paddle.py` | Handles paddle creation and movement |
| `ball.py` | Manages ball movement and bounce mechanics |
| `scoreboard.py` | Tracks and displays player scores |

---

## Gameplay

1. Two players control paddles on opposite sides  
2. The ball continuously moves across the screen  
3. Players bounce the ball using paddles  
4. Missing the ball gives the opponent a point  
5. Game continues with increasing speed and competitive gameplay  

---

## Controls

| Key | Action |
|-----|--------|
| `W` | Left Paddle Up |
| `S` | Left Paddle Down |
| `↑` | Right Paddle Up |
| `↓` | Right Paddle Down |
